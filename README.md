# UTF-Library
このライブラリは"UTF-8"・"UTF-16"・"UTF-32"の相互変換をサポートします。

## Sample

```cpp
// 使い方の一例  
  const char32_t* u32char=U"こんにちは！";  
  printf("%s",utf8(u32char));  
```
